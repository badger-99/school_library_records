# 📖 School Library Records <a name="about-project"></a>

**School Library Records** is an app that keeps track of books in a school library, and who has borrowed them.

### Key Features <a name="key-features"></a>

- **Add new students or teachers.**
- **Add new books.**
- **Save records of who borrowed a given book and when.**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

- Clone this project.
- Open it in VS code.
- Run `bundle install` to install the dependencies

### Prerequisites

In order to run this project you need

- [Ruby 3.2 or later with devkit](https://rubyinstaller.org/downloads/)
- [Ruby linters](https://github.com/microverseinc/linters-config/tree/master/ruby) - Mainly for consistent styling.

### Setup

Clone this repository to your desired folder:

> https://github.com/badger-99/school_library_records.git

### Install

This project does not need to be installed.

### Usage

Use it to maintain and update school library records.
